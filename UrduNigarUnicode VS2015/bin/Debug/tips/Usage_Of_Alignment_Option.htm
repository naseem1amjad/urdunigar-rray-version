Usage of Alignment options.

 

Summary: If “Auto Alignment” is selected, written text automatically gets aligned on change of language [Urdu / English] , other wise if “Auto Alignment” is not selected, the user needs to press “Align Left” or “Align Right” option for setting of alignments.

 

Following is the screen showing the “Auto Alignment” option as unchecked (i.e. not selected).

 

The two {Align Left & Align Right} buttons are visible when “Auto Alignment” option is unchecked (i.e. not selected). Now the user has option to press “Align Left”  or “Align Right”  button and set alignment of the text accordingly.

 

Following screen shows “Auto Alignment” option selected from menu.

 

After setting “Auto Alignment” option as checked. You can see that when ENGLISH language is selected, the text is automatically Aligned to Left.

 

 

Similarly, after setting “Auto Alignment” option as checked. You can see that when URDU language is selected, the text is automatically Aligned to Right.
